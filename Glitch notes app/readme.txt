description

this application contains an index.html file,a style.css file that contains all the styles for index.html , style2.css  contains all the styles that are applied to the text.html file 

text.html file  is responsible for displaying the create a note page, index.js contains all the code used to add text to the text area for writing notes

installation

in order to use this app keep all the files in one directory and only open the index.html file
